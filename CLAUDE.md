# CLAUDE.md

This file mirrors the repository routing model for Claude-style entrypoints.

Start with:

1. `AGENTS.md`
2. `docs/ai/project-overview.md`
3. `docs/roadmap.md`
4. The smallest relevant file under `docs/ai/modules/`
5. `docs/operations/session-handoff.md` only when recent unfinished work matters

Use `docs/` as the durable knowledge base for this repository instead of keeping a second long-form instruction layer here.
