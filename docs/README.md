# Docs Index

This repository keeps durable project knowledge under `docs/`.

## Layout

- `docs/ai/`
  - shared AI knowledge base
  - overview, workflow, module docs, and reference notes

- `docs/roadmap.md`
  - current project status
  - completed, next, later, and deferred work

- `docs/operations/`
  - operator procedures
  - session handoff notes

- `docs/architecture.md`
  - broader code and runtime structure

## Suggested Read Paths

### For humans

1. `README.md`
2. `docs/roadmap.md`
3. `docs/architecture.md`

### For agents

1. `AGENTS.md`
2. `docs/ai/project-overview.md`
3. `docs/roadmap.md`
4. one or two relevant module docs under `docs/ai/modules/`
5. only then a matching handoff or operations file
